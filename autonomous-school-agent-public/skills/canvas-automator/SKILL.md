# Canvas Automator

Create high-quality visual materials for your homework using Canva.

## Features
- **Template Search**: Find the best design for a specific assignment type.
- **Auto-Fill**: Insert text and data from your Schoology assignments into designs.
- **Exporting**: Download final designs in PDF or PNG for submission.

## Authentication
- Uses the alternate credential vault: `poopdogisgreat@gmail.com`.

## Usage Example
"Create an infographic for my Energy Systems assignment on Canva."
1. Log into Canva using alternate credentials.
2. Select an "Infographic" template.
3. Populate with energy conservation data.
4. Download and prepare for Schoology submission.
