# Precision Clicker Engine

Bypass unstable CSS selectors by finding elements via text, landmarks, or visual similarity.

## Features
- **Semantic Clicking**: Find buttons by their visual label rather than arbitrary class names.
- **Visual Stability**: Locates UI elements based on proximity to other stable landmarks.

## Usage
"Click the 'Edit' button in the OneDrive iframe."
1. Scans the UI for text labels.
2. Performs a precise coordinate-based click.
