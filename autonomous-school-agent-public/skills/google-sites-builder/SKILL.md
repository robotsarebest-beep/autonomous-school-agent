# Google Sites Builder

Automate the construction of professional portfolio websites on Google Sites.

## Features
- **Layout Selection**: Programmatically choose templates and structures.
- **Content Injection**: Insert text, images, and documents into specific site sections.
- **Publishing**: Handle the final publish and sharing settings.

## Workflow
1. Access Google Sites.
2. Select a blank or portfolio template.
3. Build pages (Biography, Career, Personal Growth).
4. Insert content crawled from Schoology or Outlook.

## Usage Example
"Build my CALM 20 portfolio website using the feedback I found."
1. Log into Google Drive.
2. Create a new Google Site.
3. Add pages for each rubric requirement.
4. Fill with generated content.
