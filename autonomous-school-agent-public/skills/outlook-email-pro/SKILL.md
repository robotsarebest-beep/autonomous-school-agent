# Outlook Email Pro

Expertly navigate and extract information from school Outlook accounts.

## Features
- **Smart Search**: Find specific feedback or instructions by teacher name or subject.
- **Attachment Handling**: Download and process assignment-related files from emails.
- **Deep Extraction**: Capture text from specific email chains for use in assignments.

## Patterns
- **Login**: Handles institutional Microsoft SSO redirects.
- **Navigation**: Uses specific selectors for the Outlook Web App (OWA).

## Usage Example
"Find the email from Mr. Grant about my interview feedback."
1. Log into Outlook via Microsoft SSO.
2. Search for "Trevor Grant".
3. Extract feedback content for the CALM 20 portfolio.
