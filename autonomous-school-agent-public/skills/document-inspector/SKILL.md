# Document Inspector

Makes template filling smarter by detecting fields, tables, and formatting requirements dynamically.

## Features
- **Field Detection**: Automatically discovers placeholders like `{{NAME}}` or specific table formats.
- **Requirement extraction**: Maps assignment rubric criteria to document sections.

## Usage
"Analyze the template for the Portfolio assignment."
1. Parses the structure.
2. Identifies where your name, contact info, and portfolio content should go.
