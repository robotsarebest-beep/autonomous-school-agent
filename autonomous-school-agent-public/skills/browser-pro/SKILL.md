# Browser Pro

Advanced navigation that allows me to bypass bot-detection systems by acting like a human.

## Features
- **Human-like Motion**: Randomizes mouse movements and typing speeds.
- **Adaptive Interaction**: Automatically detects when a page requires a human touch (CAPTCHA, 2FA, redirects) and prompts for assistance.
- **Session Persistence**: Maintains consistent cookies and browser state across sessions to prevent repeated logins.

## Usage
"Browse to the site and navigate like a human."
1. I enable the `browser-pro` navigation mode.
2. I simulate realistic user delays and interactions to navigate successfully.
