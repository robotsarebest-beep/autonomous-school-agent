# File System Orchestrator

Maintain a perfectly organized workspace for your homework.

## Features
- **Auto-Sorting**: Moves downloads from `Downloads/` to `hwbot_resources/{course}/{assignment}/`.
- **Naming Conventions**: Renames files for easy identification (e.g., `Tutorial_1_Screenshot_1.png`).

## Usage
"Organize my latest downloads for CALM 20."
1. Scans `Downloads/`.
2. Sorts files into the correct course directories.
