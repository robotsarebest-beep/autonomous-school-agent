# Persona Adapter

Transforms the agent from a literal tool into an intuitive, human-like collaborator.

## Features
- **Context Memory**: Remembers past interactions to provide proactive help.
- **Tone Tuning**: Adjusts language style to match the user's current environment (formal for school, casual for planning).
- **Adaptive Initiative**: Learns when to ask for clarification versus when to autonomously execute.

## Usage
"Start acting more like a senior engineer, less like a robot."
1. The skill triggers a re-calibration of the internal system prompt.
2. The agent adopts a more proactive, opinionated tone for the rest of the session.
